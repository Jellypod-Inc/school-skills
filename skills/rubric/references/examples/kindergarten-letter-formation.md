# Rubric: Kindergarten Letter Formation

**Assignment:** Practice writing your letters! Use the lined paper and write each letter three times.

**Alignment:** Picture rubric · 3 criteria x 3 levels · qualitative · "I can…" phrasing

| Criterion | Trying :cloud: | Getting There :sun_with_face: | I've Got It :star2: |
|---|---|---|---|
| **I can form my letters** | Letter shapes are unclear. | Most letters look right. | All my letters look right. |
| **I can stay on the lines** | Letters float above or below. | Most letters sit on the line. | All my letters sit on the line. |
| **I can space my letters** | Letters run into each other. | Some letters have space. | All my letters have space. |

**Teacher notes (not shared with student):**

- Read each row aloud with the student as self-assessment: "Show me where you stayed on the line." Student can point to examples.
- Use stickers, stamps, or smiley stamps instead of written feedback.
- No numeric score. Use this rubric formatively — track progression across weeks.
- For students working on specific letters (Orton-Gillingham or similar), adapt the first row to name the target letter(s).
- If a child shows all three smiley-faces consistently across a week, introduce a stretch goal (lowercase, then words).

**Variant: For dictation (CVC words):**

| Criterion | Trying :cloud: | Getting There :sun_with_face: | I've Got It :star2: |
|---|---|---|---|
| **I hear the sounds** | I wrote 0-1 sounds. | I wrote most sounds. | I wrote all the sounds. |
| **I wrote the letters** | Letters are unclear. | Most letters look right. | All letters look right. |
| **I used a space** | No space between words. | Some spaces. | Space between each word. |
