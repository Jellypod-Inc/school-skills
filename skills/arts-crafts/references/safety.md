# Safety Catalog

Practical age-gating for craft materials. Load only the sections you need.

## Scissors by type

| Type | Minimum age | Notes |
|---|---|---|
| Blunt safety scissors (plastic-bladed) | 3+ with close supervision | Tears paper more than cuts. Fine for snipping strips. |
| Kid scissors (rounded metal tip) | 5+ | Standard classroom scissors for K-2. |
| Pointed-tip kid scissors | 7+ | Use when cutting curves and shapes. |
| Sharp craft / adult scissors | 8+ supervised, 10+ independently | For cardboard, fabric, multi-layer cuts. |
| Rotary cutters, craft knives, X-Acto | Adult only | Never hand to a child. |

If a user says "no scissors", substitute tearing, hole-punch, pre-cut shapes, washi-tape strips, or sticker shapes. See `substitutions.md`.

## Hot glue

- Always adult-operated, including low-temp guns — low-temp still burns.
- Flag every step using hot glue as **"Adult only"** in Safety Notes.
- For any project where the child would naturally need hot glue, substitute tacky glue or white school glue and let it dry overnight.

## Small parts (choking hazard)

- The CPSC small-parts rule: anything that fits inside a cylinder 1.25" diameter × 2.25" deep is a choking hazard for children under 3.
- Common offenders: pony beads, small pom-poms (<1"), googly eyes (any size), small buttons, small dried pasta, sequins, glitter, bottle caps, small magnets.
- **Magnets**: never for under 8 — swallowed magnets cause severe intestinal injury.
- For ages 2-4, substitute: large wooden beads (≥1.5"), large pom-poms (1.5"+), sticker "eyes", construction-paper shapes.

## Edible crafts and allergens

Default to non-food crafts. When the user explicitly asks for edible (salt dough, pasta necklaces, peanut-butter bird feeders, Rice Krispie sculptures, marshmallow sculptures, edible playdough):

- Flag common allergens in Safety Notes: **peanut, tree nut, wheat/gluten, dairy, egg, soy, sesame**.
- Offer swaps inline:
  - Peanut butter → sunflower-seed butter (SunButter) or soy butter.
  - Wheat pasta → gluten-free pasta or beads.
  - Dairy yogurt → coconut or oat yogurt.
  - Egg → aquafaba or flax egg.
  - Marshmallow (gelatin) → vegan marshmallow.
- Remind: never serve craft food if it has been handled extensively or left at room temperature.

## Paints

| Type | Minimum age | Washable? | Notes |
|---|---|---|---|
| Washable tempera | 3+ | Yes | Default for all classroom + preschool painting. |
| Washable watercolor | 3+ | Yes | Low mess. Great for PreK. |
| Finger paint | 2+ | Yes | Confirm non-toxic / ASTM D-4236 label. |
| Acrylic | 6+ | No — stains | Flag staining in Safety Notes. Use smocks. |
| Oil paint / solvent-based | 12+ | No | Avoid in school and home craft contexts. |
| Spray paint | Adult only | No | Outdoors, ventilated. |

## Adhesives

| Type | Minimum age | Notes |
|---|---|---|
| Glue stick | 2+ | Default adhesive for PreK. Low mess. |
| White school glue (Elmer's) | 4+ with supervision, 5+ independently | Bottle control is hard before 5. |
| Tacky glue / craft glue | 5+ | Stronger hold than school glue; still washable. |
| Double-sided / washi / masking tape | 3+ | Good no-glue alternative. |
| Hot glue | Adult only | See above. |
| Super glue, epoxy, rubber cement | Adult only | Solvents. |
| Mod Podge | 5+ with smocks | Washable before drying. |

## Sewing needles

| Type | Minimum age | Notes |
|---|---|---|
| Plastic tapestry needle (large eye, blunt tip) | 5+ with supervision, 6+ independently | Pair with plastic canvas or burlap. |
| Metal blunt-tip tapestry needle | 7+ | For felt and aida cloth. |
| Standard sewing needle (sharp) | 8+ with supervision | 1:1 for first-time sewers. |
| Sewing machine | 10+ with supervision | Adult threads the machine. |

## Clay and oven-bake

- Air-dry clay (Crayola, Das): 3+. Non-toxic. Dries in 24-48 hrs.
- Oven-bake polymer clay (Sculpey, Fimo): 5+ (shaping only). **Adult operates oven.** Cool fully before handling.
- Homemade salt dough: 3+. Non-toxic but high salt — not for eating.
- Playdough (store-bought or homemade): 2+.
