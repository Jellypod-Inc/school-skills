# Changelog

All notable changes to School Skills are documented here.

## 0.1.0-alpha.2 — 2026-04-14

- Added `rubric`, `arts-crafts`, `lecture-to-study-guide`, and `concept-map` skills.
- All V1α skills updated with inference heuristics, clarifying-question templates, and micro-examples.
- Removed safety/copyright boilerplate from skills — handled natively by the model.

## 0.1.0-alpha.1 — 2026-04-14

- Initial V1α release with `lesson-plan`, `socratic-tutor`, and `circle-time` skills.
- Claude Code plugin manifest (`.claude-plugin/plugin.json`).
- Self-hosted marketplace catalog (`.claude-plugin/marketplace.json`).
- `npx school-skills install` installer for copying skills into `~/.claude/skills/`.
